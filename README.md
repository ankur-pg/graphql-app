## Description

This Project uses [Nest](https://github.com/nestjs/nest) framework and GraphQL.

## Dependencies
```bash
$ npm i --save @nestjs/graphql graphql-tools graphql apollo-server-express @nestjs/apollo
```

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
